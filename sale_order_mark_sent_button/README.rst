=====================================
"Mark as Sent" button for Sale Orders
=====================================

* Removes the core functionality where printing a draft automatically moves a Sale Order into the Quotation Sent state
* Replaces it with a "Mark as Sent" button that is clicked manually 

Installation
============
* Just install this module

Configuration
=============
* No configuration needed
