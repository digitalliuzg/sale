========================================
Sales Timesheet - Track Service Manually
========================================

* By default when ``sale_timesheet`` is installed, all new service products
  are suggested to be tracked with "Timesheets on Project". This can cause
  new projects to be generated by accident
* This module changes the default suggestion to be "Manually set quantities
  on order"
